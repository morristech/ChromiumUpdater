ChromiumUpdater by mgee

An updater for the OSX version of Chromium (with changelog display).

ChromiumUpdater is an updater for the OSX version of Chromium.
It displays your locally installed version and remotely available version of Chromium.
In addition it features a changelog display to track the latest changes.
